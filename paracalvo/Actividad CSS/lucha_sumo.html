Robots de sumo

Un robot de sumo es un vehículo autónomo diseñado para empujar a otro robot fuera de un área circular (llamada dojo), sin salirse él mismo del área.

El robot usa sensores para detectar al oponente y sensores de borde para no salirse del círculo. Cuando encuentra al oponente, se lanza hacia él para empujarlo. Si detecta el borde, retrocede o gira.

Tipos de sumo

  • Nano Sumo

    Tamaño max ~2.5x2.5 cm Peso max 25 g Características Muy pequeño, muy rápido

  • Mini Sumo

    Tamaño max 10x10 cm Peso max 500 g Características Más común para Arduino

  • Mega Sumo

    Tamaño max 20x20 cm Peso max 3 kg Características Más potente, más difícil

  • RC Sumo

    Tamaño max Variable Peso max Variable Características Controlado a distancia
